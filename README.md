# us-stock-lab
这是一个学习美股数据分析和量化交易的 Python 项目。
## 🚀项目目标
* 获取美股历史/实时数据，如Alpaca
* 计算常用的技术指标，如RSI，MACD
* 探索简单的交易策略
* 实现股票的动态监测和分析
## 环境准备
* Python： 3.14.4
* 主要库：。。。
## 📂 目录结构
* `/data`: 存放下载的数据。
* `/notebooks`: 存放分析用的 Jupyter Notebook。
* `/scripts`: 存放可执行的 Python 脚本。
